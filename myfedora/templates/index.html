TurboGears 2 is fast web app construction with a strong Model/View/Controller (MVC) architecture.

  1. Code your data model

    Design your data model, Create the database, and Add some bootstrap data.

    (It's easy. SQLAlchemy turns your Python objects into SQL tables. You work with Python (the gold standard in modern languages), SQLAlchemy handles the magic (base metals of SQL querying, for nearly any SQL database).

  2. Design your site architecture

    Decide your URLs, Program your controller methods, Design your templates, and Place some static files (CSS and/or JavaScript).

    (Easy again. All requests run through your Controller methods, where security is the first order of business. Generate a Python dictionary of useful data and pass it to your display templates. The Genshi template system renders it all in perfect XHTML -- and the templates play nicely with your web designer working with HTML layout software.)

  3. Distribute your app

    Test your source, Generate project documents, Build a distribution.

    (All easy. From the very quickstart your app is ready to be turned into a Python egg for easy distribution to other servers or for open source release. Unit testing is integrated and ready to use. Just add documentation and go.)
