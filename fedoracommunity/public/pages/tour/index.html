Fedora Community can help you answer these questions and more.

Package Details

Access a catalog of all Fedora packages across all active Fedora releases, chock full of useful details per package. Changelogs, version overviews, build and update statuses, downloads, source, and more!

People Details

Easily access a list of packages your friends own or maintain. Quickly look up a package maintainer's contact information to communicate with them about an issue and find out their time zone.

Search Fedora

Need to find a list of all the packages related to a particular application? Not sure what the package name is for a particular application? Know someone's IRC nick but not their name? Find out!
